# Русский перевод субтитров для серии видео от Xiph.Org

Этот репозиторий содержит перевод оригинальных английских субтитров видео "[Episode 2: Digital Show & Tell](https://www.xiph.org/video/vid2.shtml)" на русский язык. Первое видео серии ("[Episode 1: A Digital Media Primer for Geeks](https://www.xiph.org/video/vid1.shtml)") уже включает русские субтритры, но в них есть неточности, которые я исправил.

## Содержимое

Исправленные русские субтритры для Episode 1:

1. [vid1-ru.srt](vid1-ru.srt)

Перевод английских субтитров (US English) на русский:

1. [02-Digital_Show_and_Tell.ru.titles.vtt](02-Digital_Show_and_Tell.ru.titles.vtt) -- субтритры на русском с переводом текста из видео (рекомендуется использовать этот файл).
2. [02-Digital_Show_and_Tell.ru.en.vtt](02-Digital_Show_and_Tell.ru.en.vtt) -- субтитры идут одновременно на русском и английском.
3. [02-Digital_Show_and_Tell.ru.vtt](02-Digital_Show_and_Tell.ru.vtt) -- только русские субтритры.
